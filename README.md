# How-to-buy-data-in-Nigeria

A non-parametric data driven guide to stop wasting money, and planning data plans according to actual usage and user pattern. By combining a rigorous analysis of market plans with insights from user's data, each user can be cost, waste and time effective in buying their data plans and not just doing it by habit.

---

### 🧠 The Philosophy of the Model: The Patterned Routine

This project operates on a core belief: **Human data usage is never truly random.** Even in the most varied lifestyles, behavior follows a "gravity" or a center. Whether a user is a structured professional or someone with a highly unpredictable schedule, patterns eventually emerge over time. 

By extracting the **mean and median** of these routines and testing them against a **rigorous, standardized analysis of market plans**, this model bypasses the "marketing noise." Instead of buying data based on habit or guesswork, we use the user's own behavioral pattern to mathematically determine the path of maximum efficiency.
